Java Gossip: TicTacToe 井字遊戲（Applet）

這個程式不是原創，我只是修改與結合而已。。。。

Applet與連線部份的程式是從Deitel & Deitel的Java How to Program中取材，原程式僅限於一對遊戲者連線，我將之改為可以多對連線，每兩個玩家連續的連線會被配成一對進行遊戲。

除了可連線對戰之外，還可以與電腦對戰，與網路上找到的Java AI package結合，Java AI package的部份並作了一些修改。

與電腦對戰部份，並沒有進行網路連線，只是將applet抓下來，然後在本機上執行。

AI的部份可以使用MiniMax法或是AlphaBeta法，從程式碼中直接改！可以設定搜尋深度，搜尋越深越難贏，最多平手吧！

Java Applet介面的程式還可以與網友進行連線對戰，您必須先啟動一個伺服器程式，您可以NetworkTicTacToe的上一層目錄使用以下的指令進行啟動：
java NetworkTicTacToe.server.TicTacToeServer
