語言技術：GTK Gossip


GTK （Gimp ToolKit）原 本來影像處理軟體 GIMP（GNU Image Manipulation Program） 的開發工具箱，後來成為一個跨平台的圖形介面開發套件，GTK 基於 C 語言，但不限於使用 C 語言，您也可以使用 C++、Java、PHP、Perl、Python、Ruby 等語言來撰寫 GTK 程式（參考 Language Bindings）， 這邊的文件將記錄一些 GTK+ 的心得，所使用的環境為 Ubuntu 8.04。
